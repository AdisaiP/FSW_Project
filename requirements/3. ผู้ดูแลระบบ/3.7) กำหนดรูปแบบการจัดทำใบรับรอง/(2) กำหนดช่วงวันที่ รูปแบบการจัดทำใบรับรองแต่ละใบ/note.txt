# (2) กำหนดช่วงวันที่ รูปแบบการจัดทำใบรับรองแต่ละใบ

URL: http://localhost:5055/Admin/CertificateTemplate

จำนวนรูปที่ต้องแคป: 3 รูป

รายการหน้าที่ต้องแคป
1. Modal #modalSetProductionMode > Tab 'กำหนดรูปแบบ'
   - section 'ช่วงวันที่มีผล': วันที่เริ่มต้น (required) + วันที่สิ้นสุด (optional)
   - dynamic preview text (alert-success): 'รูปแบบนี้ใช้งานได้ตั้งแต่ [วันที่] ถึง [วันที่/ไม่มีกำหนด]'
2. Modal > Tab 'ประวัติรูปแบบการจัดทำ'
   - ตาราง ช่วงวันที่/รูปแบบ (badge)/ผู้กำหนด/หมายเหตุ/สถานะ
   - แถว 'มีผล' highlight table-success + badge 'ใช้งานอยู่ตั้งแต่ [วันที่]' มุมขวาบน
3. ตารางหลัก + badge 'ใช้งานอยู่ตั้งแต่ [วันที่]' (สีเขียว) ต่อแถว template
