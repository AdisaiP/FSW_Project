# (5) เพิ่ม/แก้ไขไฟล์ลายมือชื่ออิเล็กทรอนิกส์ สำหรับแต่ละหน่วยงาน

URL: http://localhost:5055/Admin/Organization
Navigation: Tab 'หน่วยงานภาครัฐ' > ปุ่มดูรายละเอียด > Tab 'ลายมือชื่ออิเล็กทรอนิกส์'

จำนวนรูปที่ต้องแคป: 4 รูป

รายการหน้าที่ต้องแคป
1. Modal #govAgencyDetailModal > Tab 'ลายมือชื่ออิเล็กทรอนิกส์' (#govTab-esign)
   - ตารางรายการ: ผู้ลงนาม/ตำแหน่ง/วันที่เริ่ม/วันหมดอายุ/สถานะ/ไฟล์/จัดการ
   - badge 'ลายมือชื่อ: ใช้งาน N รายการ' / 'ใกล้หมดอายุ' ใน modal header
2. ปุ่ม 'เพิ่มลายมือชื่อ' > modal #modalAddOrgESign
   - select ผู้ลงนาม + upload drop zone (PNG/SVG 500KB) + file preview + ช่วงวันที่ + หมายเหตุ
3. ปุ่ม 'แก้ไขไฟล์' ต่อแถว > modal #modalEditESignFile
   - security notice + เปรียบเทียบ ลายมือชื่อเดิม vs ใหม่ + เหตุผล (required) + วันที่มีผล
4. badge 'ยังไม่มีไฟล์' / 'มีไฟล์' ต่อแถว + icon เตือน ⚠️ ใกล้หมดอายุ
