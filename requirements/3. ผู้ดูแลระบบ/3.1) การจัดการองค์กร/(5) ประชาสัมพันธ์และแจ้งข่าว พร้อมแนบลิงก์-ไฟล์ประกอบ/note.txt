# (5) ประชาสัมพันธ์และแจ้งข่าว พร้อมแนบลิงก์-ไฟล์ประกอบ

URL: http://localhost:5055/Admin/Announcement

จำนวนรูปที่ต้องแคป: 4 รูป

รายการหน้าที่ต้องแคป
1. หน้าหลัก Admin/Announcement
   - summary cards (ข่าวทั้งหมด/ใช้งาน/หมดอายุ) + ตาราง + filter + ปุ่ม 'เพิ่มข่าว'
2. Modal 'เพิ่ม/แก้ไขข่าวประชาสัมพันธ์' (#addAnnouncementModal)
   - หัวข้อข่าว TH/EN + เนื้อหา (rich text) + แนบลิงก์ + upload ไฟล์ + ช่วงวันที่แสดง
3. Modal 'รายละเอียดข่าว' (#annDetailModal)
   - Badge ประเภท/สถานะ + เนื้อหาครบถ้วน + ไฟล์แนบ + วันหมดอายุ
4. Modal 'ตัวอย่างการแสดงผล' (#previewModal)
   - Preview หน้าจอที่ผู้ประกอบการ/เจ้าหน้าที่จะเห็นใน Dashboard
