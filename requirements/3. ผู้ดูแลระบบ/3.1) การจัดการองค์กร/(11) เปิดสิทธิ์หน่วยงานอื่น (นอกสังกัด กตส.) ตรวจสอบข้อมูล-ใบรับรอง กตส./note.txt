# (11) เปิดสิทธิ์หน่วยงานอื่น (นอกสังกัด กตส.) ตรวจสอบข้อมูล-ใบรับรอง กตส.

URL: http://localhost:5055/Admin/Organization

จำนวนรูปที่ต้องแคป: 4 รูป

รายการหน้าที่ต้องแคป
1. Admin/Organization > Tab 'หน่วยงานภาครัฐ' > badge 'ภายนอก: 2 หน่วยงาน'
   - summary badge + ปุ่ม 'เพิ่มสิทธิ์หน่วยงานภายนอก'
2. #govAgencyDetailModal > Tab 'สิทธิ์หน่วยงานภายนอก'
   - ตาราง 2 หน่วยงาน (ชื่อ / ประเภทสิทธิ์ / วันที่ / สถานะ)
3. Modal #grantExternalAccessModal
   - select หน่วยงาน + checkbox ประเภทข้อมูลที่ให้เข้าถึง + ช่วงวันที่ + เหตุผล
4. Alert ยืนยัน + ปุ่ม 'บันทึกและแจ้ง Admin'
