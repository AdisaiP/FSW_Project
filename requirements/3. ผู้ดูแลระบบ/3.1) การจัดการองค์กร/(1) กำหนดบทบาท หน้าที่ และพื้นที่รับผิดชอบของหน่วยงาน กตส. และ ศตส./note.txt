# (1) กำหนดบทบาท หน้าที่ และพื้นที่รับผิดชอบของหน่วยงาน กตส. และ ศตส.

URL: http://localhost:5055/Admin/Organization

จำนวนรูปที่ต้องแคป: 5 รูป

รายการหน้าที่ต้องแคป
1. หน้า Admin/Organization > Tab 'หน่วยงานภาครัฐ (กตส./ศตส.)'
   - ตารางรายการหน่วยงาน + summary cards + filter + ปุ่ม 'เพิ่มหน่วยงาน'
2. Modal 'เพิ่ม/แก้ไขหน่วยงาน' (#govAgencyAddModal)
   - radio กตส./ศตส. + select จังหวัด + checkbox บทบาท + ชื่อ TH/EN
3. Modal #govAgencyDetailModal > Tab 'ข้อมูลหน่วยงาน'
   - accordion ข้อมูลทั่วไป (ชื่อ, ที่อยู่, ประเภท กตส./ศตส.)
4. Modal > Tab 'พื้นที่รับผิดชอบ'
   - chips จังหวัดที่รับผิดชอบ + ตาราง RMBS1 ในพื้นที่
5. Modal > Tab 'บทบาทหน้าที่'
   - ตารางบทบาทหน้าที่ (ออกใบรับรอง/อนุมัติ/ตรวจสอบ)
