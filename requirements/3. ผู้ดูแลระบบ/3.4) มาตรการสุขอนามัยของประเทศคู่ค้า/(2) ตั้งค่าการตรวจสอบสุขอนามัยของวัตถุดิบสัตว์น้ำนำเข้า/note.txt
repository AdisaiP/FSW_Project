# (2) ตั้งค่าการตรวจสอบสุขอนามัยต่อมาตรการ

URL: http://localhost:5055/Admin/NotificationReason

จำนวนรูปที่ต้องแคป: 3 รูป

รายการหน้าที่ต้องแคป
1. Detail modal (#measureDetailModal) > Tab 'ตั้งค่าการตรวจสอบ' > accordion 'ความถี่การตรวจ'
   - radio รายปี/รายไตรมาส/รายเดือน + วันที่ตรวจครั้งถัดไป
2. accordion 'เงื่อนไขการแจ้งเตือน'
   - แจ้งก่อน N วัน + checkbox ช่องทาง (ระบบ/อีเมล/SMS) + ปุ่ม 'ดูตัวอย่าง toast'
3. accordion 'โรงงานที่ได้รับผลกระทบ'
   - ตารางโรงงาน DOF List ที่เกี่ยวข้อง + badge จำนวน + ปุ่ม 'เพิ่มโรงงาน'
