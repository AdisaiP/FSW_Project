# (3) ตรวจสอบรายละเอียดมาตรการสุขอนามัย

URL: http://localhost:5055/Admin/NotificationReason

จำนวนรูปที่ต้องแคป: 5 รูป

รายการหน้าที่ต้องแคป
1. Detail modal > Tab 'ข้อมูลหลัก' > accordion 'ข้อมูลทั่วไป'
   - รหัสมาตรการ/ประเทศ/ภูมิภาค/ขอบเขตสินค้า/หน่วยงานรับผิดชอบ/สถานะ badge
2. accordion 'เงื่อนไขและการกำกับ'
   - วันมีผลบังคับ/ทบทวนครั้งถัดไป/ระดับความเสี่ยง/วัตถุประสงค์/หน่วยงานต้นทาง/หมายเหตุ
3. Tab 'ข้อกำหนดหลัก' + badge นัยสำคัญ (สูง=danger/กลาง=warning/ต่ำ=success)
4. Tab 'เอกสารอ้างอิง' + ตาราง (เอกสาร/เวอร์ชัน/วันที่ประกาศ/สถานะ)
5. Tab 'ประวัติการเปลี่ยนสถานะ' (แสดง badge สถานะเดิม > ใหม่)
