# (1) ปรับแบบฟอร์มอิเล็กทรอนิกส์ เมื่อประเทศปลายทางแก้ไขกฎหมาย/รูปแบบใบรับรอง

URL: http://localhost:5055/Admin/CertificateTemplate

จำนวนรูปที่ต้องแคป: 4 รูป

รายการหน้าที่ต้องแคป
1. หน้า CertificateTemplate + badge 'เวอร์ชันปัจจุบัน' + ปุ่ม 'แก้ไขรูปแบบ' ต่อแถว
2. Modal แก้ไขรูปแบบ > Tab 'แก้ไขข้อมูล'
   - ฟิลด์ใหม่: เหตุผลการปรับ (select) + วันที่มีผลบังคับใช้ (required) + อ้างอิงเอกสาร + หมายเหตุ
   - checkbox 'เพิ่มเลขเวอร์ชัน (Bump Version)' + approval alert
3. Tab 'ประวัติการแก้ไข'
   - ตาราง เวอร์ชัน/วันที่/เหตุผล/ผู้แก้ไข/ผู้อนุมัติ/สถานะ (Activated=success/รอดำเนินการ=warning)
4. Alert 'การเพิ่มเวอร์ชันต้องการการอนุมัติก่อน Activate' + ปุ่ม 'ส่งอนุมัติ'
