# (5) สร้าง-แก้ไข-ระงับ-ยกเลิก + ประวัติการดำเนินการ + Export Excel

URL: http://localhost:5055/Admin/DofList

จำนวนรูปที่ต้องแคป: 4 รูป

รายการหน้าที่ต้องแคป
1. ตารางหลัก + action column: ปุ่ม ดูรายละเอียด/แก้ไข/ระงับ/ยกเลิก/พิมพ์
2. Modal #dofSuspendModal: เหตุผลระงับ + วันที่มีผล
3. Modal #dofCancelModal: เหตุผลยกเลิก + alert 'ไม่สามารถกู้คืนได้'
4. Detail modal > Tab 'ประวัติการเปลี่ยนสถานะ' + Export Excel dropdown (4 ตัวเลือก)
