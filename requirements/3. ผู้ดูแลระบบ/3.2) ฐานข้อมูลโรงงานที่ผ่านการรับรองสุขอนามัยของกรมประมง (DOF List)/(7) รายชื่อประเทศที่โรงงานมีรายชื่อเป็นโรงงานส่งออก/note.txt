# (7) รายชื่อประเทศที่โรงงานมีรายชื่อเป็นโรงงานส่งออก

URL: http://localhost:5055/Admin/DofList

จำนวนรูปที่ต้องแคป: 3 รูป

รายการหน้าที่ต้องแคป
1. คอลัมน์ 'ตลาดส่งออก' badge chips ในตารางหลัก + filter dropdown 'ประเทศ'
2. Detail modal > Tab 'ประเทศที่รับรองส่งออก' + ตาราง (ประเทศ/กลุ่มตลาด/สถานะ/วันที่)
3. ปุ่ม 'เพิ่มประเทศ' + badge กลุ่มตลาด (EU/JP/US/OTHER) + badge สถานะ
